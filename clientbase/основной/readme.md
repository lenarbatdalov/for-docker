# используемые технологии
```
apache2.4
php7.2
mysql5.7
adminer

Стандартный порт 80 прокинут на 8080.
http://localhost:8080/
Дефолтный виртуальный хост, с включенными модулями:
	rewrite
	headers

До-установленные php модули:
	gd
	imap
	mysqli + pdo-mysql
	zip
	tidy
	xmlrpc
	bcmath
	exif
	memcached
	opcache
	imagick

Также установлен xdebug и настроен для работы в ide phpstorm:
	порт - 9000
	server name - localhost
	ide key - PHPSTORM
```
